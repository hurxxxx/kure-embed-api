문서 변환 테스트

개요

이 문서는 datalab-marker를 사용한 문서 변환 기능을 테스트하기 위한 샘플 문서입 니다.

주요 기능

  • HTML에서 마크다운으로 변환 •
  • 이미지 추출 및 저장 •
  • 다양한 출력 형식 지원 (markdown, json, html) •
  • 한국어 텍스트 처리 •

지원 형식

입력 형식 설명
PDF Adobe PDF 문서
DOCX Microsoft Word 문서
PPTX Microsoft PowerPoint 프레젠테이션
HTML 웹 페이지 문서

더 복잡한 수식: ∫₀^∞ e^(-x²) dx = √π/2
